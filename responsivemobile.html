Mobile friendly and mobil first

Mobile friendly

Refers to disigning first for desktop first and working the way down to smaller devices

Mobile first

Start designing for small screens first. Principals for mobile first:

Graceful Degradation

Start with a full feartured desktop centric website, remove and rearrange and restyle content for smaller viewports, decrease bandwith usage (remove large images and videos). Add touch centric features.

  • designing for modern browsers first
  • Provide fallbacks for unsupported features
  • Ensure basic functionality works for older browsers

Progresive Enhancement

Focus on the best experience for mobile first, reduce reliance on mouse actions, load content quickly and esasily for small screens and determine the essentials at the beginning
  • Focus on content and accesibility first
  • Create base-level experience first
  • Add advance features for enhancements

Creating Flexible layouts

Use percentage base widths for page components, use min max widths to add constraints to the layout while maintaining flexibility, and use percentage based width and min max values together

Media Queries

Media query consist of two components: media type and media features

Media types

  • print: matches to printers and other print related displays
  • speech: matches screen readers
  • screen: matches all devices that are not print or speech
  • all: matches to all devices

Media features

@media (min-width: 769px) {} /* larger than 768px (most landscape tablets and computers*/)
